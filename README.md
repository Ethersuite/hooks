# NestPack Hooks

A NestJs hooks system for developing extendable module logic.

**Features**:

- .env file support
- programmatic overrides and defaults
- process.env support
- constants autogeneration

## Installation

```bash
$ npm install @nestpack/hooks
# OR
$ yarn add @nestpack/hooks
```
